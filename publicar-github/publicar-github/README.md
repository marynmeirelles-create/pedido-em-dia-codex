# Atelie em Dia

Aplicativo offline para organizar pedidos, clientes, agenda e backups do atelie.

## Publicacao

Envie todos os arquivos desta pasta para o repositorio do GitHub.

Para publicar pelo GitHub Pages:

1. Abra o repositorio no GitHub.
2. Envie estes arquivos para a branch principal.
3. Va em Settings > Pages.
4. Em Source, escolha Deploy from a branch.
5. Selecione a branch principal e a pasta raiz.
6. Salve e aguarde o link de publicacao.

O app funciona sem servidor e salva os dados no proprio navegador da usuaria.
